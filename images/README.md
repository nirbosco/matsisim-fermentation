# תמונות לאתר

האתר מחפש את הקבצים הבאים בתיקייה הזו. כל עוד הם חסרים, מוצג פלייסהולדר אפור עם תיאור. ברגע שתכניסו קובץ בשם המתאים, הוא יופיע אוטומטית. פורמט מומלץ: JPG, רוחב 1200px ומעלה.

| `hero.jpg` | תמונת נושא: שולחן עם כמה צנצנות התססה צבעוניות (לאורך, יחס ~4:5) |
| `history.jpg` | שוק ירקות / צנצנות ישנות / מדף כבושים (יחס ~16:10) |
